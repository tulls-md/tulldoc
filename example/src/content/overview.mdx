# Пример документации

`@tulls-md/tulldoc` — платформа для создания сайтов документации на базе Next.js.

Напишите `.mdx`-файлы, положите их в папку — tulldoc соберёт сайт с боковой навигацией,
оглавлением страниц и красивой вёрсткой. Никаких шаблонов, никакой лишней конфигурации.
