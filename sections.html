Разделы

Раздел — это ссылка в шапке, которая переключает боковое меню на свой набор страниц. Так документацию можно разбить на независимые ветки: например «Начало работы» с гайдами и «Компоненты» со справочником — у каждой свой сайдбар.

Эта документация сама использует разделы: вкладки «Текстовые документы» и «Автодокументирование» в шапке меняют боковое меню.

Как объявить раздел

Раздел — это папка верхнего уровня, перечисленная в поле sections корневого meta.json:

{
  "pages": ["overview", "getting-started"],
  "sections": [{ "name": "mdx", "label": "Текстовые документы" }, "doc"],
  "header": [
    "changelog",
    { "label": "GitHub", "href": "https://github.com/acme/ui" }
  ]
}

Элемент sections — это либо имя папки строкой, либо объект { name, label }:

ФормаПодпись вкладки в шапке
"doc"title из doc/meta.json, иначе имя папки в PascalCase
{ "name": "doc", "label": "…" }значение label

Порядок элементов в массиве задаёт порядок вкладок в шапке. Внутри папки-раздела URL уходит глубже: страница mdx/files.mdx доступна по /mdx/files.

Раздел по умолчанию

Файлы в корне content/ (не входящие ни в одну папку-раздел) образуют раздел по умолчанию — он показывается первым. По умолчанию его вкладка подписана «Документация»; чтобы переименовать, задайте title в корневом meta.json:

{
  "title": "Начало работы",
  "pages": ["overview", "getting-started"],
  "sections": ["mdx", "doc"]
}

В примере выше раздел по умолчанию — это overview и getting-started.

Активный раздел определяется по URL: если адрес начинается с папки-раздела — активен он, иначе — раздел по умолчанию.

Чем раздел отличается от группы и от шапки

  • Группа — папка верхнего уровня, которой нет в sections. Она остаётся подзаголовком внутри текущего сайдбара (см. Файлы и навигация). Папка становится разделом только при явном указании в sections.
  • header — по-прежнему перечисляет одиночные страницы и внешние ссылки (GitHub, npm), которые показываются в шапке рядом с переключателем разделов. На таких страницах сайдбар скрыт.

Внутри папки-раздела вложенные папки работают как обычно — становятся группами бокового меню этого раздела.
